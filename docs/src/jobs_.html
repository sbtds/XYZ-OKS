お仕事

求人情報の登録・管理

求人情報は「お仕事」カスタム投稿タイプで管理されています。 通常の投稿と同じような操作で求人情報を追加・編集できます。

1 新規求人の追加

管理画面の「お仕事」→「新規追加」をクリックして、 新しい求人情報を登録します。以下の項目を入力してください：

  • タイトル: 求人のタイトル（例：営業職募集）
  • 本文: 詳細な仕事内容
  • アイキャッチ画像: 企業ロゴや職場の写真
  • カテゴリー: 職種カテゴリーを選択
  • 勤務地: タグで勤務地を設定

2 求人詳細情報の入力

編集画面の下部にある「求人詳細」セクションで、 以下の情報を入力します：

  • • 給与: 月給25万円〜35万円
  • • 雇用形態: 正社員 / 契約社員 / パート
  • • 勤務時間: 9:00〜18:00（実働8時間）
  • • 応募資格: 必須スキルや経験
  • • 福利厚生: 社会保険完備、交通費支給など

※すべての項目を入力することで、求職者により詳しい情報を提供できます。

3 公開設定と管理

公開状態を設定します：

  • 公開: サイトに表示されます
  • 下書き: 管理者のみ確認可能
  • 非公開: 募集終了した求人

重要: 募集が終了した求人は削除せず、「非公開」に変更してください。 データを保持することで、再募集時に簡単に再公開できます。

よくある質問

求人の表示順を変更したい

投稿日時を変更することで表示順を調整できます。 「クイック編集」から日付を変更してください。

特定の求人を目立たせたい

「この投稿を先頭に固定表示」にチェックを入れると、 常に一覧の最上部に表示されます。
